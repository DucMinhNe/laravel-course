# Exercise — Lesson 02

Match each item to its location in a Laravel 10 app:

| You want to… | Path |
| --- | --- |
| Add a `PostController` | `app/Http/Controllers/PostController.php` |
| Define the `/posts` URL | `routes/web.php` |
| Write the posts page template | `resources/views/posts/index.blade.php` |
| Change the default DB connection | `config/database.php` (+ `.env`) |
| Find today's log | `storage/logs/laravel.log` |

**Task:** open a fresh app and verify each path exists (or is where you'd
create the file). Bonus: run `php artisan about` and read the environment
summary it prints.
